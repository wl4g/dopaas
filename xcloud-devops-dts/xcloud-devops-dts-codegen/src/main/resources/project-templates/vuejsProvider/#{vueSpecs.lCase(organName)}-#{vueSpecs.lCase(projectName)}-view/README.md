### Quick start

```
npm install

# Running project, Serve with hot reload at localhost:8090
npm run dev

# Build project, Build for production with minification
npm run build
```

### Project describe:
- /src 主要编辑代码
- /build 打包配置代码
- /node_modules 依赖包文件夹
- /static 某些静态文件
- /config 生产环境和开发环境下的配置
- /config/index.js 可配置开发环境的端口,代理地址等
